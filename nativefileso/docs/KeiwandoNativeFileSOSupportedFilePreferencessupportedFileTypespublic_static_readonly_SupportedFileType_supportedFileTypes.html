Keiwando.NativeFileSO.SupportedFilePreferences

static property

supportedFileTypes

public static readonly SupportedFileType[] supportedFileTypes

Summary

The file types that should be associated with the mobile versions of the application.

Remarks

Users are going to be able to open files of the file types included in this array from outside of the application, e.g. using the native "Share" and "Open in" functionality within Android and iOS.
