Keiwando.NativeFileSO.SupportedFileType

instance property

AppleConformsToUTI

public string AppleConformsToUTI

Summary

Set this value if you are declaring a custom file type that conforms to an existing UTI. Separate multiple UTIs with a pipe (|).

Remarks

For example, if you are declaring a custom file type based on plain-text, you should set this property to "public.plain-text".
