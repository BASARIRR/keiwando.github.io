Keiwando.NativeFileSO.NativeFileSOMobile

FilesWereOpened

public event Action FilesWereOpened;

Summary

Occurs when files were opened without the user being manually prompted to open files from within the app. This occurs if certain file types have been associated with the application. See SupportedFilePreferences for more information on file associations.
