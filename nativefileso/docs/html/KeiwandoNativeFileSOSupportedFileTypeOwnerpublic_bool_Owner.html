Keiwando.NativeFileSO.SupportedFileType

instance property

Owner

public bool Owner

Summary

Specifies whether this application is the primary creator of this file type.

Remarks

This property is only added to the Info.plist file for iOS projects when associating this SupportedFileType with the application. It can be ignored if iOS is not a targeted platform.
