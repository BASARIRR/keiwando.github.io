Keiwando.NativeFileSO.NativeFileSOMacWin

instance property

SelectOpenPaths

public void SelectOpenPaths (SupportedFileType[] fileTypes, bool canSelectMultiple, string title, string directory, Action onCompletion)

Summary

Presents a native dialog to the user which allows them to select multiple file paths for opening files.

Remarks

On macOS, this will present a non-floating panel.

Parameters

fileTypes

The file types used to filter the available files shown to the user.

canSelectMultiple

Determines whether mutliple files can be selected and loaded at once.

title

Title of the shown dialog. Note: The title is not shown for macOS > 10.11

directory

The default directory in which file navigation should start. If this value is empty, the panel will remember the last visited directory.

onCompletion

A callback for when the presented dialog has been dismissed. At this point, files have either been opened or the selection has been cancelled.
