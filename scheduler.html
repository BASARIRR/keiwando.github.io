Commands

Select a Day

Type Monday, Tuesday, Wednesday, Thursday, Friday, Saturday, Sunday or Today to select a day

The currently selected day is always highlighted.

You can also abbreviate all day names down to a minimum of 2 characters, so for example Mo, Mon, Mond, Monda, Monday are all valid ways to reference Monday.

Add an Event

Specify the event in the form

Day startTime - endTime Title

Add !! to the end of the command to indicate an important event. For example

Wed 10:30 - 12 Meeting !!

adds an important event called "Meeting" on Wednesday between 10:30 and 12:00 to your schedule.

Times

There are many different ways to specify a time. You can choose to just write the hours, e.g. 8 means 8:00 o'clock.

If you want to also specify minutes you can use a colon, comma or a dot: 1:30 1,30 1.30 are all the same.

You can also write 2pm instead of 14:00 if you like.

Specify a time range by writing the start and end time separated by either - or to. With a time range, you only need to write PM once and it will be interpreted for both the start and end input time if that makes sense. For example, 2 to 3pm means 14:00 - 15:00, but 10 to 3pm is interpreted as 10:00 - 15:00.
